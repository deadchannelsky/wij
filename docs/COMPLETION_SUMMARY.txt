Common Issues & Solutions:
  • No calendar events?
    → Make sure calendar is public
    → Verify API key is correct
    → Check sync status endpoint

  • HTTPS not working?
    → Run deploy.sh again to validate
    → Check certificate: openssl x509 -in /etc/letsencrypt/...
    → Check Nginx: sudo tail -f /var/log/nginx/whereisjason_error.log

  • Service won't start?
    → Check logs: sudo journalctl -u whereisjason -n 50
    → Verify .env is readable
    → Check database exists: ls -la /opt/whereisjason/server/data/app.db
